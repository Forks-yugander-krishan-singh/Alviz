Basic Commands for 'Turtle Canvas'
forward(distance)
_-_ move Turtle forward distance pixels
left(angle)
_-_ turn Turtle left angle degrees
right(angle)
_-_ turn Turtle right angle degrees
clear()
_-_ clear the canvas but keep Turtle's current position
reset()
_-_ reset Turtle to original state
hideTurtle()
_-_ hide the Turtle
showTurtle()
_-_ show the Turtle
goto(x, y)
_-_ move the Turtle to (x, y) coordinate
write(string)
_-_ draw a string on canvas on Turtle's current position
wrap(boolean)
_-_ wrap the line when Turtle moves out of the boundary of canvas
penup()
_-_ lift the Turtle up -> no drawing when calling forward(distance)
pendown()
_-_ set the Turtle down
