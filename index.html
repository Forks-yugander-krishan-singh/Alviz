What's the fuss about...

Alviz is a fun way to learn algorithm and visualization in Javascript. It is pre-loaded with Javascript libraries to support drawing in Turtle mode or Artist mode. Read more below to learn how to use the commands.

Help Wanted: I found it fascinating on recursive algorithms (a.k.a 'recipes') which make beautiful arts such as fractals, snowflakes, etc. If you have an interesting recipe to share (ie, recursively drawing a cat), please send me a pull request or email me at hoangdangninh[at]gmail[dot]com.

Note: Make sure to turn off Vimium (Chrome) or Vimperator (Firefox) if you are an avid Vim user.

Turtle Mode
tspeed(speed = null)
:: set Turtle's drawing speed
treset()
:: reset Turtle's mode
tclear()
:: erase all drawings but keep Turtle's position
tforward(distance)
:: move Turtle forward a distance number of pixels
tleft(angle)
:: turn Turtle left an angle degree
tright(angle)
:: turn Turtle right an angle degree
thide()
:: hide Turtle
tshow()
:: show Turtle
theading(obj)
:: set obj.angle to Turtle's angle
tsetheading(obj)
:: set Turtle's angle to obj.angle
tgoto(x, y)
:: move Turtle to (x, y) coordinate
tsetpos(obj)
:: move Turtle to (obj.x, obj.y) coordinate
tpos(obj)
:: set (obj.x, obj.y) to Turtle's current position
twrite(text)
:: draw text on canvas at Turtle's current position
twrap(boolean)
:: set canvas boundary when Turtle moves out of bound
tpenup()
:: lift Turtle up, drawing invisible
tpendown()
:: put Turtle down, drawing visible
tlog(obj)
:: show obj information
tcsize()
:: show Turtle canvas size
Artist Mode
areset()
:: reset Artist's mode
acv
:: HTMLCanvasElement canvas object
actx
:: CanvasRenderingContext2D object
Extras
download(url)
:: embed external javascript library to be used with Alviz
