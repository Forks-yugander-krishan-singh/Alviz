What's the fuss about...

AlgoFun is a fun way to learn algorithm visualization in Javascript. It is pre-loaded with Javascript libraries to support drawing in Turtle mode or Artist mode. Read more below to learn how to use the commands.

Disclaimer: Not all algorithms can be easily expressed visually but many of them do. The author of AlgoFun currently likes to focus on RECURSIVE algorithms and try to express each one of them in an artistic way. If you share the same interest or would like to contribute, please send him your Algo as a pull request ;)

Note: Make sure to turn off Vimium (Chrome) or Vimperator (Firefox) if you are an avid Vim user

Turtle Mode
resetT()
:: reset Turtle to original state
clear()
:: clear the canvas but keep Turtle's current position
forward(distance)
:: move Turtle forward distance pixels
left(angle)
:: turn Turtle left angle degrees
right(angle)
:: turn Turtle right angle degrees
hideTurtle()
:: hide the Turtle
showTurtle()
:: show the Turtle
goto(x, y)
:: move the Turtle to (x, y) coordinate
write(string)
:: draw a string on canvas on Turtle's current position
wrap(boolean)
:: wrap the line when Turtle moves out of the boundary of canvas
penup()
:: lift the Turtle up -> no drawing when calling forward(distance)
pendown()
:: set the Turtle down
Artist Mode
resetA()
:: reset all canvases to be like original
cv
:: [object HTMLCanvasElement] is the canvas object in Artist mode
ctx
:: [object CanvasRenderingContext2D] is the 2D context object in Artist mode
new Chart(ctx, chart)
:: render a chart, read more at chartjs documentation
